# 🚴 Plan de Entrenamiento Mensual

Repositorio para gestionar y automatizar el plan de entrenamiento mensual basado en evidencia científica.

## Estructura

```
📁 planes/          → Planes mensuales en Markdown
📁 registros/       → Registro diario de sesiones
📁 metricas/        → CSVs con métricas de rendimiento
📁 .github/workflows/ → Automatizaciones
```

## Modalidades
- 🚴 Ciclismo (commute + entrenamiento)
- 💪 Fuerza / Gym
- 🏃 Elíptica / Cardio
- 🧖 Sauna (recuperación)

## Automatizaciones activas
- Recordatorio semanal de actualizar el plan (lunes 7:00 AM)
- Resumen mensual automático al final de cada mes
